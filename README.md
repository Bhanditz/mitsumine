mitsumine
=========
Mitsumineははてなブックマークを閲覧するためのAndroidアプリです。  
はてブを快適に流し見するために自分用に開発しています。  
同時に、kiriminが新しいライブラリやAPIを試すためのSandBox的なアプリでもあります。
